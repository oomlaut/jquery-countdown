jquery.countdown.js
===================



## Usage:

<pre>
$('#countdown_days').countdown({
    end: new Date("January 11, 2014"),
    format: "%d days, %h:%m:%s",
    afterEventMessage: 0
    });
</pre>

Loosely based on

[jquery-utils](http://code.google.com/p/jquery-utils/wiki/CountDown)  
[javascript kit snippet](http://www.javascriptkit.com/script/script2/count2.shtml)
